# Facebook Wow Reaction using HTML & CSS | Facebook Reaction Emojis

Um projeto simples de **reação estilo Facebook (Wow Emoji)** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar uma **animação de reação estilo Facebook**, com emoji interativo e efeitos visuais.

## 🛠️ Tecnologias

- HTML
- CSS (animation, keyframes)

## 🎯 Funcionalidades

- Emoji de reação (Wow)
- Animação interativa
- Design criativo e divertido

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
